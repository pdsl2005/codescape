# CODESCAPE

> Welcome to Codescapes, where you build cities from your code. This opensource project was built by members of Northeastern University's premier ideation club, *Forge* 

## Highlights

Codescapes will be baked into your repository as an extension of VSCode. This extension is intended for use with <prog language TBD>

Features: 

- Interactive cities of your code
- Buildings created by the functionality of your code and methods
- Exportable cities in HTML/React
